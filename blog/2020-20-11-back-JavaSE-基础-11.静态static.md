## static
- 当类加载时,static变量、变量 也一并加载被加载出来了。
- 静态变量(类变量)
- 静态方法
	- 方法内部: 可以调用 其他静态方法
	- 方法内部: 无法使用类的非静态方法(非静态方法: 当类被实例化时才存在的)
- 静态代码那块
	- 只会被执行一次
	- 在构造器被调用之前执行


<!--truncate-->


```java
public class Student {
	static {
		System.out.println("静态代码快")
	}

	public Student() {
		System.out.println("狗仔其")
	}
}
```
