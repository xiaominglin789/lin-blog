##  java 优势
- 面向对象
- 高性能、分布式、多线程
- 安全性、健壮性

## java中 JDK、JRE、JVM 关系
- JDK: Java Development Kit (java)
- JRE: Java Runtime Environment (运行时环境)
- JVM: Java Virtual Machine (虚拟机)
JDK 包含了： JRE 和 JVM
