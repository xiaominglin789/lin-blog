## 抽线类(约束规则类)
- abstract 修饰符修饰的类 -> 抽象类
- abstract 修饰方法 -> 抽象方法,只负责声明,不写实现
- 抽象类不能 `new`,只能靠子类去实现
- 抽象方法必须在抽象类中，抽象类有普通方法的声明和实现。

```java
// 抽象类
public abstract class Action {
	// 抽象方法声明,子类可以去实现
	public abstract void doSomeThing();
}
```
